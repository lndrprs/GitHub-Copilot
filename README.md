C1.1 - Introdução
 > - GitHub Copilot é um assistente de AI para codificação;
 >   - O assistente ajuda a escrever código e com menos esforço - permitindo o humano focar em resolução de problemas e colaboração; 
 >   - Há uma pesquisa informando que o Copilot melhora a produtividade do desenvolvedor, e acelera o desenvolvimento de software. 
 > - Copilot pode ser usado para:
 >   - Sugestões de códigos enquanto digita na IDE; 
 >   - Ajudar no seu código;
 >   - Pedir ajuda usando a linha de comando;
 >   - Organizar e compartilhar contexto com o Copilot Spaces, para ter mais respostas relevantes;
 >   - Gerar descrições e mudanças em um Pull Request;
 >   - Trabalhar em mudanças de códigos e criar Pull Request para você revisar (Apenas Pro+, Business e Enterprise). 
 >
 > - Copilot pode ser usado em:
 >   - IDE;
 >   - GitHub Mobile;
 >   - Windows Terminal Canary;
 >   - Linha de Comando;
 >   - Site do GitHub. 

C1.2 - Planos 
 > - Free: Acesso limitado;
 > - Pro: Mais flexível - Conclusões limitadas, modelos premiuns, Coding agent, concessão mensal de requisições premiums;
 > - Pro+: Maior level de acesso para desenvolvedores individuais, acesso total a modelos no Chat e maior concessão de requisições;
 > - Business: Agente de código, e permite gerenciamento centralizado, e controle de políticas para os membros da organização;
 > - Enterprise: Possui todas as características do Business, e permite gerenciar organizações, ou usuários e times diretamente. 
 >
 > - Maiores informações: https://docs.github.com/en/copilot/get-started/plans
